# crypto-price-cli
A command line tool to get current market price of cryptos. It scrapes the price from Coindesk website.

## Install package from npm 
```bash
    # using npm
    npm i crypto-price-cli
```

## Usage
```bash
    # Get <crypto> price
    crypto-price-cli <crypto>

    # Get crypto list
    crypto-price-cli -l
```
